# Class Management System 🏫

This is a basic web-based class management project developed using HTML, CSS, and JavaScript. It allows teachers or admins to manage class-related data such as student records, attendance, and announcements.

## 🌟 Features
- Add and view student information
- Track attendance manually
- Easy-to-use interface for quick updates
- Fully responsive design for desktop and mobile

## 🚀 How to Use
1. Clone or download the repository.
2. Open the `index.html` file in your browser.
3. Use the navigation to manage students and classes.

## 🔧 Technologies Used
- HTML5
- CSS3
- JavaScript
